# 方法论差距分析 — 2026-03-24

> 基于 14 篇参考研报/论文（Barra CNE5、Fama-French、长江证券纯因子系列、东方证券因子选股系列、国泰君安组合优化）与 quant-dojo 当前方法的对比。

## 核心判断

quant-dojo 当前处于**"朴素多因子打分"**水平。专业 A 股量化策略的标准做法是**"结构化风险模型 + 组合优化"**。差距不在代码基础设施（已经很好），而在**方法论**。

---

## Top 3 必做改进（预计夏普从 0.34 → 0.8+）

### 1. 行业中性化（影响最大，预计夏普 +0.3~0.5）

**当前问题：** 完全没有行业控制。选出的 30 只股票可能集中在同一行业，导致 2018 年整个行业暴跌时回撤 -45%。

**研报结论：** 东方证券 #17 证明，仅控制行业因子就能把最大回撤从 -6.2% 降到 -4.0%，几乎不损失 alpha（5.0% → 4.9%）。行业是 A 股最重要的风险因子。

**具体做法：**
- 选股时按行业分组，每组选 Top N
- 或在组合优化中约束行业 active exposure = 0

### 2. 组合优化替代等权（预计夏普 +0.2~0.4）

**当前问题：** Top 30 等权 + 10% cap。30 只太集中（个股风险大），等权无法控制风格暴露。

**研报结论：** 国泰君安论文用 MVO + 风格中性约束实现了 IR 4.21、最大回撤 -2.1%。东方证券 #6 证明即使简单的启发式行业内分配也比等权好。

**具体做法：**
- 扩大持股数到 100-200 只（降低个股风险）
- 用 MVO（均值方差优化）+ 约束：行业中性、风格中性、个股上限
- 工具：CVXPY 求解凸优化问题

### 3. 扩大因子库（预计夏普 +0.2~0.3）

**当前问题：** 只有 3 个有效因子（reversal、low_vol、turnover），全是技术/统计因子，零基本面因子。

**研报结论：** Fama-French 证明 book-to-market 是最稳健的截面预测因子。国泰君安论文中 Earnings Yield 纯因子 IR = 2.47。标准 A 股因子库有 20-70 个候选因子。

**具体做法：**
- 优先加入：EP（盈利收益率）、BP（市净率倒数）、ROE、营收增速
- 数据已有（`utils/fundamental_loader.py`）
- 用 Fama-MacBeth 回归验证显著性后再纳入组合

## 中期改进（夏普 0.8 → 1.5+）

### 4. 结构化风险模型 + 协方差矩阵

**当前问题：** 没有风险模型，无法做组合优化。

**做法：** 构建 Barra 式风险模型 Σ = X·F·X' + Δ，用 EWMA + Newey-West 估计因子协方差。

### 5. Fama-MacBeth 因子验证框架

**当前问题：** 只用 IC/ICIR 验证因子，不控制因子间多重共线性。

**做法：** 每期做截面回归 R = B·f + ε，t 统计量 > 2 才算有效。VIF > 3 的因子剔除。

### 6. 风格中性化（size、volatility）

**当前问题：** 没有控制市值和波动率暴露。

**做法：** 选股前将 alpha 因子对 log(市值) 做截面回归取残差（市值中性化）。

### 7. 纯因子组合（长江证券方法）

**当前问题：** z-score 合成混淆了因子间的相关性。

**做法：** 用 WLS 截面回归（权重 = sqrt(市值)）估计纯因子收益。

## 当前方法 vs 专业做法对比

| 维度 | quant-dojo 当前 | 专业做法 | 差距 |
|------|----------------|---------|------|
| 因子数量 | 3 个 | 10-20 个 | 严重不足 |
| 因子类型 | 纯技术 | 技术+基本面+分析师 | 缺基本面 |
| 因子验证 | IC/ICIR | FM 回归 + 8 项 BARRA 标准 | 不够严格 |
| 因子合成 | z-score + IC 加权 | WLS 纯因子回归 | 方法粗糙 |
| 风险模型 | 无 | Barra CNE5（10 风格 + 30 行业） | 完全缺失 |
| 行业控制 | 无 | 行业中性约束 | 最大短板 |
| 组合构建 | Top 30 等权 | MVO + 约束（100-200 只） | 方法落后 |
| 持股数 | 30 | 100-200 | 集中度过高 |
| 协方差估计 | 无 | EWMA + Newey-West | 缺失 |
| 绩效归因 | 无 | Brinson/因子归因 | 缺失 |

## 建议执行顺序

| 阶段 | 内容 | 预计效果 | 复杂度 |
|------|------|---------|--------|
| 1 | 行业中性化 | 回撤从 -45% → -20% | 中 |
| 2 | 加 EP/BP/ROE 基本面因子 | 年化 +3-5% | 中 |
| 3 | 持股扩到 100 只 | 回撤再降 5-10% | 低 |
| 4 | MVO 组合优化（CVXPY） | 夏普 +0.3 | 中高 |
| 5 | FM 回归因子验证 | 剔除伪因子 | 中 |
| 6 | 结构化风险模型 | 全面风控 | 高 |
