### Phase 1 统计基础：`02_statistics_basics.ipynb` ✅

| 章节 | 内容 | 核心结论 |
|------|------|---------|
| 1. 收益率分布 | 直方图 + 理论正态曲线 + QQ 图 + Shapiro检验 | A 股收益率有胖尾，非正态（p<0.05 拒绝正态） |
| 2. t 检验 | 单样本t检验（均值是否为零）+ 双样本t检验（策略 vs 基准） | 双均线策略收益大概率统计不显著 |
| 3. 相关性分析 | 8只代表性股票相关性热力图 + 滚动60日相关性 | A 股个股高相关（均值 ~0.5），危机时更高 |
| 4. 截面 IC | 动量因子（20日）Rank IC 时序 + 累积IC | 短期动量因子 IC 接近零，A 股短期无明显动量 |
| 5. 自相关 | 自相关柱状图 + Ljung-Box 检验 + 不同时间尺度动量vs反转 | 日收益率基本无序列相关，波动率有聚集效应 |

## 技术问题 & 解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| `from utils.xxx import` 报找不到模块 | 没有 editable install，notebook 的工作目录不在项目根 | 创建 `pyproject.toml` + `pip install -e .`，同时在 notebook 首行 `sys.path.insert(0, "../../..")` |

## 2026-03-21 Phase 3 因子研究完成

### 做了什么

**完成 Phase 3 因子研究阶段，交付 4 大经典因子框架：**

1. **价值因子** (Value Factor)
   - EP（盈利收益率）、BP（账面收益率）、SP（销售收益率）等权合成
   - 文件：`research/factors/value/value_factor.py + 06_value_factor.ipynb + README.md`
   - 预期 IC：0.02~0.06，ICIR：0.3~0.8

2. **质量因子** (Quality Factor)
   - ROE、ROE 稳定性（滚动 8 期标准差）、毛利率等权合成
   - 文件：`research/factors/quality/quality_factor.py + 07_quality_factor.ipynb + README.md`
   - 前视偏差处理：shift(1) + ffill 对齐到日频
   - 预期 IC：0.03~0.08，ICIR：0.3~0.8

3. **低波动因子** (Low Volatility Factor)
   - 已实现波动率（20 日滚动 std）、Beta（60 日滚动 OLS）、等权合成
   - 文件：`research/factors/low_vol/low_vol_factor.py + 08_low_vol_factor.ipynb + README.md`
   - 低波动异象理论支撑完整

4. **因子分析工具完善**
   - `factor_decay_analysis()`：计算不同持有期的 IC 衰减规律
   - `ic_weighted_composite()`：动态权重多因子合成
   - `neutralize_factor()`：行业 + 市值中性化
   - `factor_summary_table()`：批量因子汇总表

### 发现了什么

**关键认识：**

1. **财务数据前视偏差**：
   - A 股季报公布滞后 1~4 个月（年报最长到次年 4 月底）
   - 直接用报告期末会引入严重前视偏差
   - 本方案用 shift(1) + ffill 缓解，但仍非完全无偏（需实际公告日时间戳）

2. **因子有效性预期差异**：
   - 动量：A 股短期反转为主，传统动量效应弱（实测 IC ≈ 0）
   - 质量：白马股溢价明显（消费/医药），但 ROE 不稳定行业较弱
   - 价值：均值回归机制存在，但周期性强（牛市表现差）
   - 低波动：A 股散户主导，低波溢价理论上弱于成熟市场

3. **数据质量陷阱**：
   - PE ≤ 0 的亏损股，PB ≤ 0 的资不抵债公司必须置 NaN
   - PS 数据当前用 pcf 替代，不精确
   - 幸存者偏差：需用历史成分股回测，非当前成分股

4. **行业暴露风险**：
   - 质量因子：集中于白马行业（银行、白酒、消费）
   - 价值因子：天然偏向金融、地产等高杠杆行业
   - 动量因子：可能追涨板块轮动噪音
   - 建议：Phase 4 必须加行业中性化
