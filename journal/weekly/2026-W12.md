### Phase 1 统计基础：`02_statistics_basics.ipynb` ✅

| 章节 | 内容 | 核心结论 |
|------|------|---------|
| 1. 收益率分布 | 直方图 + 理论正态曲线 + QQ 图 + Shapiro检验 | A 股收益率有胖尾，非正态（p<0.05 拒绝正态） |
| 2. t 检验 | 单样本t检验（均值是否为零）+ 双样本t检验（策略 vs 基准） | 双均线策略收益大概率统计不显著 |
| 3. 相关性分析 | 8只代表性股票相关性热力图 + 滚动60日相关性 | A 股个股高相关（均值 ~0.5），危机时更高 |
| 4. 截面 IC | 动量因子（20日）Rank IC 时序 + 累积IC | 短期动量因子 IC 接近零，A 股短期无明显动量 |
| 5. 自相关 | 自相关柱状图 + Ljung-Box 检验 + 不同时间尺度动量vs反转 | 日收益率基本无序列相关，波动率有聚集效应 |

## 技术问题 & 解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| `from utils.xxx import` 报找不到模块 | 没有 editable install，notebook 的工作目录不在项目根 | 创建 `pyproject.toml` + `pip install -e .`，同时在 notebook 首行 `sys.path.insert(0, "../../..")` |
| `setuptools.backends.legacy` not found | pyproject.toml 写了错误的 build-backend | 改为 `setuptools.build_meta` |
| akshare 下载速度慢 | 单线程串行 | `batch_download` 用 `ThreadPoolExecutor`，并发4线程 + 自动重试 |
