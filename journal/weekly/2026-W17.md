### 发现了什么

- **v2 spec §9 item 3 自己留的 out ("若 BB 单腿优于 ensemble 则独立 pre-reg BB-only") 真的触发了** — pre-reg discipline 起作用, 不是事后改口径

- **config-driven 架构优于硬编码**: 之前 ENSEMBLE_MIX = 0.5 是 module-level 常量, 如果 v2/v3 共存期 (e.g. 不同 portfolio_dir 跑不同 spec) 会冲突; 现在 trader 构造时注入, 一个 process 可同时有 v2 / v3 两个实例

## 2026-04-22 周三 (凌晨) — 差异化因子挖掘 (RIAD + MFD)

Issue #33. 依用户要求重点看 2025 年数据, 尝试 2 个 A 股独有因子.

### 做了什么

1. **数据审计**: 基于 `journal/tushare_data_inventory_20260420.md`,
   确认 `ths_hot` / `dc_hot` / `stk_surv` / `moneyflow` 等未被已有 ~20 因子利用.
2. **RIAD 因子** (Retail-Institution Attention Divergence, `research/factors/retail_inst_divergence/`):
   - 散户热度 (ths_hot + dc_hot 榜单 rank 加权) 减去 机构调研机构数 (log1p).
   - cross-section zscore 相减. window: retail 20d, inst 60d.
   - FULL IC=-0.070, ICIR=-0.89, HAC t=-5.16
   - **OOS 2025: IC=-0.059, ICIR=-1.25, HAC t=-13.30, pct_pos=8.9%**
   - size-neutralize 后 corr=0.996 且 IC 略增强 → 不是小盘代理
   - 分层 OOS Q1-Q5 差 +1.09% (每 20 日持有)
3. **MFD 因子** (MoneyFlow Divergence, `research/factors/moneyflow_divergence/`):
   - elg (超大单) 净流入 ratio 减去 sm (小单) 净流入 ratio. 归一化除以总成交额.
   - FULL IC=-0.020, HAC t=-4.28, pct_pos=36.6% → **反转因子**
   - 解读: elg 不等于 smart money, 很可能是尾盘对倒 / 派发假象
4. **RIAD+MFD 合成**: IC=+0.069, 与 RIAD 单独 0.070 几乎无差别 → MFD 无增量 alpha
5. **FMD 设计待命**: `research/factors/foreign_margin_divergence/factor.py` 写好, 但
   `northbound/000001.parquet` 2025 仅 5 行, 数据缺失, 暂不能用. 待 tushare 补齐或换接口.

### 发现了什么

- **RIAD 是一个真实、稳健、A 股独有、2025 有效的 cross-sectional alpha**:
  - 非 size 代理 (相关度 0.996 但 IC 略增强)
  - 主要 alpha 在做空端 (散户追捧股跑输), IS Q5=-1.10%/期, OOS Q5=+2.00% vs Q1=+3.09%
  - ICIR ≥ 0.85, 方向高度一致 (周 IC 只有 9% 是正的)
- **"大单净流入 = bullish"是错的**: tushare elg 分类有噪音, 但反过来 "大单净流入多反而跑输"
  在 5.5 年长样本上稳健, 说明市场微观结构里 informed selling 伪装成大单买入普遍存在
- **合成不总能增强**: 当两个因子 mechanism 相似 (都是 smart/dumb money divergence),
  叠加反而分散 (MFD 弱信号拖累强 RIAD)
- **northbound 数据在 2024-2025 严重稀疏** (每股每年几乎只有几十行), 需要重新下载或换源

## 2026-04-22 凌晨 (下半场) — Option C: 扩展到 6-factor 组合

jialong 选 Option C 后继续挖 #4~#6 因子.

### 做了什么

1. **LULR** (连板反转, `research/factors/limit_up_ladder/`)
   - 基于 tushare.limit_list up_stat + open_times + limit 类型
   - 5 日事件驱动 LS, 1449 events
   - FULL Sharpe -1.57 (长期亏损), 但 2024+ regime 反转有效 (OOS 2025 ls Sharpe +0.63)
   - 成本 drag 高 (每事件 0.6%), 实盘不可行, 仅可作 regime indicator
2. **THCC** (筹码集中度, `research/factors/top_holder_concentration/`)
   - 基于 top10_floatholders 季度环比变化, 两变体 all vs inst
   - **反直觉**: 机构加仓反而 bearish, OOS 2025 thcc_inst Sharpe **-1.71**
   - 真因: 季报滞后 1.5 月, 公告日信息已 price-in, follow 即追高; window dressing
   - 关键洞察: **'smart money follow' 假设仅对实时信号 (RIAD 调研/BGFD 金股) 有效, 滞后季报不适用**
3. **SB** (调研 burst, `research/factors/survey_burst/`)
   - 7d / 91d median 时序 spike, 与 RIAD inst leg (绝对水平) 区分
   - Null effect: OOS 2025 IC +0.001, HAC t +0.13
   - 启示: **机构关注 level 有 alpha, spike 无 alpha** (event 驱动调研已被快速 price-in)
4. **6-factor orthogonality matrix** (`research/factors/_orthogonality_v2.py`)
   - 全部 |corr| < 0.3 = 正交
   - 最大 pair: MFD-LULR 0.27; 其次 RIAD-SB 0.20, RIAD-MFD 0.19
   - BGFD / THCC 对全谱 |corr| < 0.1, 最独立

### 发现了什么

- **唯一过 IC 门槛 (>0.03) 的仍是 RIAD (0.070)**. 其他 5 个全部 |IC| ≤ 0.02
- **正交 ≠ 有 alpha**, IC-weighted ensemble 下 RIAD 权重 0.7+, 合成 ≈ RIAD 单独
- **Regime shift 2024 分界** 被 LULR 独立验证 (2019-2023 连板反转不成立, 2024+ 成立)
- **'Follow smart money' 的细分**:
  - 实时信号 (RIAD 调研, BGFD 月度金股) ✅ 正 alpha
  - 滞后披露 (THCC 季报) ❌ 反而负 alpha
  - 时序 spike (SB) 则 null

结论 (给 jialong): **Option A 仍是最优**, 维持原建议. 补充:
- BGFD / LULR → universe filter 候选 (入榜 ∩ RIAD 打分 / 避雷池)
