### 发现了什么

- **v2 spec §9 item 3 自己留的 out ("若 BB 单腿优于 ensemble 则独立 pre-reg BB-only") 真的触发了** — pre-reg discipline 起作用, 不是事后改口径
- **PV 腿是稀释剂不是分散器**: BB 4/5 + PV 1/5 → ensemble 2/5. correlation matrix 没体现 ensemble dilution, 只有 full-stack 5-gate 才能看出
- **bootstrap CI_low 对样本 variance 非常敏感**: 8-yr BB SR 0.84 (ann 16% / std 19%), bootstrap std(SR) ≈ 0.32 → CI_low 直接掉到 0.20. 要过 0.5 需要 live SR 稳定 ≥ 1.0 超过 6 mo
- **Insider 2018 tail risk**: 单年 ann -45.8%, 没发现很多 drawdown control 能救 — alpha 本身薄
- **config-driven 架构优于硬编码**: 之前 ENSEMBLE_MIX = 0.5 是 module-level 常量, 如果 v2/v3 共存期 (e.g. 不同 portfolio_dir 跑不同 spec) 会冲突; 现在 trader 构造时注入, 一个 process 可同时有 v2 / v3 两个实例
