# v11 ICIR + 波动率自适应止损 — 评估报告 — 2026-04-16

## 方法

- 因子权重：训练期 20 日前向收益计算 ICIR，权重 ∝ |ICIR|，min_weight 5%
- 自适应止损：`threshold_t = -8% × clip(σ_t / σ_ref, 0.5, 2.0)`
  - σ_t = 60 日滚动年化波动率
  - σ_ref = 训练期 σ_t 中位数（IS 整段：**0.2206**；WF 每窗口独立计算）
  - 阈值区间：[-4%, -16%]
- 无未来泄漏：ICIR cutoff + ref_vol 都严格限于训练期

## 核心发现：自适应机制被"阉割"了

### v11 OOS ≡ v10 OOS（小数点后 4 位一致）

| 指标 | v11 OOS | v10 OOS | 差 |
| --- | ---: | ---: | --- |
| 年化 | +5.98% | +5.98% | 0 |
| 夏普 | 0.2749 | 0.2749 | 0 |
| 超额 | -11.10% | -11.10% | 0 |
| 最大回撤 | -17.76% | -17.76% | 0 |

这不是 bug——是自适应机制在 2025 OOS 期间**完全退化为固定阈值**。

### 根因分析

1. **IS σ_ref = 0.2206**（10 年长期锚点）
2. **2025 OOS 的 σ_t 大多在 0.18~0.25 之间**，非常接近 σ_ref
3. `ratio = σ_t / σ_ref ≈ 1.0` → `threshold = -8% × 1.0 = -8%`（= v10 固定阈值）
4. 结果和 v10 100% 相同

### 更深层诊断：问题不是"高波动"而是"短促急跌"

2025 的 OOS 崩盘源于几次**短促（3-5 天）方向性急跌 + 快速反弹**的"假摔"行情：

- 60 日滚动波动率会**把这些脉冲 smoothed 掉**，σ_t 始终接近长期均值
- 止损以回撤触发（-8%），但这些快速急跌恰好能踩到 -8% 然后反弹
- 降仓 50% 后，需要净值创新高才恢复满仓——在震荡市里可能永远到不了

**自适应机制的假设（"2025 是高波动环境"）本身就错了**。真实问题是短期价格路径的尖锐度，不是波动率水平。

### 为什么 v10/v11 的 IS 结果却好看？

IS 2015/2018 的大熊市是**持续数月的趋势性下跌**（高波动+方向明确），
这种场景下 -8% 止损触发后能避开后续持续下跌，所以 IS 回撤从 -42% → -24%。

**一刀切的回撤止损只对"慢跌"有效，对"脉冲"无效。** 这解释了 IS 与 OOS 的巨大反差。

## 结论

### **REJECT v11** — 自适应未解决问题

v10 和 v11 OOS 完全相同，证明：
1. 60 日滚动波动率对 2025 的脉冲型急跌不敏感
2. 自适应机制在 2025 期间被夹到 ratio ≈ 1，退化为固定阈值
3. 问题本质不在"波动率自适应"这个维度

## 连续两个止损实验的启示

| 版本 | 机制 | IS 结果 | OOS 结果 |
| --- | --- | --- | --- |
| v10 | 固定 -8% | -42% → -24% ✅ | -11% 超额 ❌ |
| v11 | 波动率自适应 | 同 v10 | **完全等同于 v10** |

**两次实验同一个结论：回撤触发的止损层对这个策略的 OOS 无效。**

应该放弃"在 v9 上面加止损"的思路，考虑这两条根本路径：

### 路线 C（新）：regime-gated 止损
借鉴 v8 的做法：**只在市场明确下行时启用止损**（比如 HS300 < 120 日均线）。
- 原因：大熊市才是慢跌趋势，止损有效；反之短期急跌就是噪音
- 代价：引入 regime 指标，但比 v8 简单（不需要 RSRS，不需要 state-per-weight 表）

### 路线 D（新）：接受 v9 + 降仓位重新平衡
把单票持仓从 30 只扩到 60-80 只 + 因子得分加权，
从源头降低组合波动率（而不是事后止损）。
- 预期：IS 最大回撤可能自然降到 -30% 附近
- 代价：可能损失一些夏普（更分散 = 信号弱化）

## 建议

**接下来 v12 优先尝试路线 D**（持仓集中度重构）。理由：
- 不引入新的 regime 识别复杂度
- 直接针对回撤源头（单票权重过大），而非事后补救
- 可以和 v9（已验证 OOS 1.60）完全叠加
- 如果 v12 仍失败，回头试路线 C

路线 C 保留作为路线 D 失败时的备选。
