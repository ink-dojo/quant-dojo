# 数据目录

> 数据文件不入 git（已在 .gitignore 配置）。本目录只存脚本和文档。

---

## 目录说明

| 目录 | 内容 | 来源 |
|------|------|------|
| `raw/` | 原始下载数据，不做任何处理 | akshare / tushare / 聚宽 |
| `processed/` | 清洗、对齐、复权后的数据 | 由脚本生成 |

---

## 数据源

### akshare（免费，推荐入门使用）
```python
import akshare as ak
# 无需注册，直接使用
```

### tushare（免费注册，积分制）
- 注册：https://tushare.pro
- 基础数据免费，高频数据需积分

### 聚宽 JoinQuant
- 研究环境免费，有历史数据回测平台
- 本地使用需要购买数据

---

## 数据规范

1. **日期格式**：统一使用 `YYYY-MM-DD`，存储为 `pd.DatetimeIndex`
2. **复权**：默认使用**前复权（qfq）**
3. **价格单位**：元（人民币）
4. **成交量单位**：股
5. **收益率**：小数形式（0.01 = 1%），不是百分比

---

## 已知数据问题

- A股有涨跌停限制，异常大收益率需检查是否为复权错误
- 退市股票数据可能不完整（幸存者偏差）
- 节假日停牌需正确处理缺失值
